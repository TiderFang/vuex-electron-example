# Vuex Electron Example

Usage example of [Vuex Electron](https://github.com/vue-electron/vuex-electron)

### How to run

```
yarn install
yarn start:dev
```

### Where to look

1.  Creation of Vuex Store:

    ```javascript
    src / store / index.js
    ```

2.  Using of store in renderer process:

    ```javascript
    src / index.js
    src / views / app.js
    ```

3.  Using of store in main process:

    ```javascript
    src / main.js
    ```
